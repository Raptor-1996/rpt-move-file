# Usage

## Basic Usage
sudo python3 rpt_move.py

## Menu Options

    Move a file - Move files with root privileges
    Check file info - Display file information
    Exit program - Close the application

## Direct Terminal Usage
You can also use it directly from terminal:

``` bash

sudo python3 rpt_move.py --source /path/src --dest /path/dest
```
